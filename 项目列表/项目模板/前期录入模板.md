# 前期录入模板

> 适用场景：项目经理在项目启动前，把活动的基础信息、模块、节点、联系人和风险一次性录入到 Golden Project。
> 建议填写方式：先填「基础信息」和「时间范围」，再填「模块」、「活动清单」、「联系人」和「风险事项」。
> Excel 版同名模板 `前期录入模板.xlsx` 是正式录入版本，这份 Markdown 主要用于阅读和校对。
